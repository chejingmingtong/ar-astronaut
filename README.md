# AstroFit AR 试衣间

这是一个手机端 AR 换装网页原型。页面会调用摄像头，用 MediaPipe PoseLandmarker 做人体关键点识别，并把宇航服绘制到人体上。模型或网络不可用时，会自动进入演示叠加模式。

## 本地预览

```powershell
cd D:\111
python -m http.server 4173
```

然后打开：

```text
http://127.0.0.1:4173/
```

## 手机扫码

手机摄像头权限通常要求 HTTPS。正式给用户扫码时，请把这个目录部署到 HTTPS 域名，打开首页后二维码会自动生成 `?ar=1` 入口；用户扫码进入后会直接隐藏二维码面板，进入 AR 页面。
